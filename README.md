第三章 名称空间
主要是做一个区分命名空间的问题
所有c++的标准库里的函数都要加std：：
如果一个代码中 我想输出cout 又想把cout做一个变量 这时候就用std：：来区分
一般编程的时候不建议函数名和变量用同一个。
using namespace std 告诉编辑器我们将要使用空间std中的函数或者对象。
